# Animated Navigation Menu Bar

Projeto desenvolvido para praticar a criação de menus de navegação animados utilizando HTML, CSS e JavaScript.

## 🚀 Objetivo

O objetivo deste projeto foi aprimorar conhecimentos em desenvolvimento front-end, animações CSS e construção de interfaces modernas e interativas para websites.

## 🛠 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## ✨ Funcionalidades

- Menu de navegação animado
- Efeitos visuais interativos
- Layout moderno
- Transições suaves
- Estrutura responsiva

## 📚 Conceitos praticados

- CSS Animations
- Hover Effects
- Transitions
- Responsividade
- Manipulação de interface
